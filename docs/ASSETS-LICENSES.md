## 设计参考（未直接引用资源）

下列开源项目仅作为**扁平插画风配色与层次**的审美参考，代码中无拷贝的 path 数据：

- **Twemoji** (CC-BY 4.0) — 通用 emoji 构图比例参考；若未来引入 Twemoji path 片段将在此追加署名。
- **OpenClipart / Kenney.nl** (CC0) — 简笔画层次结构参考。

## 语音可编辑性

每个 sprite 使用 `palette` 槽位着色（默认 2–5 色）。用户说「把房子改成蓝色」时，
L0/L1 通过 `modify.set.fill` 更新 `palette[0]`，素材重绘时整体换色。

## 禁止项（本 PR 已遵守）

- ❌ NC / ND / CC-BY-SA 素材内嵌
- ❌ unDraw 等禁止再分发的素材库
- ❌ 运行时 CDN 拉取外链图片（素材库）
- ❌ 位图贴图作为 sprite（仅 `image` 节点用于 Seedream 可选增强）
